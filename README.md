## このリポジトリについて
このリポジトリにあるプラグインは、[一角ジェミニ](https://ci-en.net/creator/8147)様が作成されたものを、一角ジェミニ様がリポジトリを削除されることとなったため、Nulluo Kikiuraが一角ジェミニ様の承諾を得たうえでリポジトリの管理を引き継いだものとなります。
